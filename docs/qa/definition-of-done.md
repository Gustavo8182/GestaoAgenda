# Definition of Done

Uma mudança está concluída somente quando:

- critérios de aceite atendidos;
- testes unitários e de integração relevantes;
- teste de isolamento multiempresa;
- autorização revisada;
- migração versionada e reversibilidade considerada;
- mensagens de erro úteis;
- build backend e frontend verdes;
- teste E2E para fluxo crítico quando aplicável;
- documentação atualizada;
- diff sem alterações não relacionadas;
- nenhum segredo ou dado sensível versionado;
- nenhuma funcionalidade excluída adicionada;
- smoke test no ambiente de validação.
