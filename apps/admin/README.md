# Painel administrativo

Aplicação Angular 22 em modo standalone e TypeScript estrito.

## Instalação

```bash
npm install
```

Na primeira instalação, versione o `package-lock.json` gerado.

## Desenvolvimento

```bash
npm start
```

## Testes

```bash
npm run test:ci
```

## Build

```bash
npm run build
```

As páginas atuais são placeholders navegáveis. Não implemente telas completas antes de a regra correspondente existir no backend e nos critérios de aceite.
